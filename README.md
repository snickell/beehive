# Oh Beehive!

Agents, flow and colab.

## Setup

Requirements: install [nvm](https://github.com/nvm-sh/nvm), or have whatever node version is in [.nvmrc(./nvmrc) already installed some other way.

```
nvm install # if you're using nvm
npm install
```

## Commands

### Start a dev server
```
npm run dev
```

### Run tests in a loop

```
npm run test
```
